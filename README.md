# SafePassAnt
*A simple password manager, pronounced like "En Passant"*

This is a simple password manager that is pretty good for personal use.

## Features

- Master Password
- Storing and Viewing of Password
- Passwords stored locally and encrypted
- Password Manager

## Made with

- C++
- CMake
- STL

## Build

How to build:

```
mkdir build
cd build
cmake ..
cmake --build .
```
